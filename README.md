# 📊 Employee Salary Data Visualization

## 📌 Description
This project is based on data visualization using Python. It uses Pandas, NumPy, and Matplotlib libraries to analyze and visualize employee salary data.

## 🚀 Features
- Data handling using Pandas
- Numerical operations using NumPy
- Visualization using Matplotlib
- Multiple chart types:
  - Bar Chart
  - Pie Chart
  - Scatter Plot

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## 📥 Installation
Install required libraries using:
pip install pandas numpy matplotlib

## ▶️ Usage
Run the Jupyter Notebook or Python file:
jupyter notebook

## 📊 Charts Included
- Bar Chart (Salary comparison)
- Pie Chart (Salary distribution)
- Scatter Plot (Salary trend)

## 📈 Output
The project displays different types of graphs for better understanding of employee salary distribution and comparison.
